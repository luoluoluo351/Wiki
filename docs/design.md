# 设计规范

> UI 样式将在第二阶段统一实现，第一阶段仅保证功能可用。

## 配色方案（暂定）

- 主色调：深色背景 + 金色点缀（修仙主题）
- 背景色：#1a1a2e
- 卡片背景：#16213e
- 强调色（金）：#e2b04a
- 文字色：#e0e0e0
- 次要文字：#a0a0a0

## 布局

- 顶部导航栏，固定 60px 高
- 内容区最大宽度 1200px，居中
- 卡牌网格：3-4 列自适应

## 卡牌样式

- 圆角 8px
- 悬停时微微上浮 + 阴影增强
- 卡片内容：头像(上方) + 信息区(下方)

## 字体

- 系统默认字体栈：`"Microsoft YaHei", "PingFang SC", sans-serif`
