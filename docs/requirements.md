# 需求文档

## 项目概述

修仙游戏 Wiki，纯 HTML+CSS+JS 本地 PC 端网页。提供角色、法宝、灵宠、功法神通等图鉴功能，以及伤害计算器工具。

## 模块需求

### 1. 角色图鉴 `#characters`

- 卡牌式网格排列，卡片显示头像+姓名+境界+灵根
- 点击卡片跳转详情页（新页面/路由）
- 自定义添加角色（表单），数据存 LocalStorage
- 详情页包含：立绘、姓名、境界、灵根、基础属性(1→100级线性成长)、进阶属性(固定值)、被动技能(可增删)、突破记录(可增删)、突破属性栏(可增删)

### 2. 法宝图鉴 `#treasures`

- 三种类型：攻击类、防具类、饰品类
- 词条系统：攻击类(基础攻击力+可选属性)、防具类(基础生命+可选属性)、饰品类(仅可选属性)
- 等级 1-60，词条值线性成长
- 品阶：下品/中品/上品/极品法器

### 3. 怪物图鉴 `#monsters` — 未开发

### 4. 道具图鉴 `#items` — 未开发

### 5. 灵宠图鉴 `#pets`

- 卡片展示：图片+名称+境界+五行标签
- 基础属性 1→60 级线性成长
- 进阶属性 1→60 级线性成长
- 主动技能和被动技能（可增删文本框）

### 6. 功法与神通 `#skills`

- 行式展示：小图+名称+五行属性+等阶
- 功法(被动) / 神通(主动) Tab 切换
- 等阶：人阶下/中/上
- 神通额外 CD 输入框
- 纯图鉴，不参与伤害计算

### 7. 剧情 `#story` — 未开发

### 8. 地图 `#map`

- 拖拽图片 + 全宽展示
- 替换/覆盖功能

### 9. 伤害计算器 `#damage`

- 攻击方手动输入属性 + 选择法宝
- 防御方手动输入属性
- 自动计算修为压制、五行克制、防御免伤
- 结果：防御免伤比例、非暴击伤害、暴击伤害
