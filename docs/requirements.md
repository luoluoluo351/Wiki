# 需求文档

## 项目概述

修仙游戏 Wiki，纯 HTML+CSS+JS 本地 PC 端网页。提供角色、法宝、灵宠、功法神通、道具等图鉴功能，伤害计算器工具，以及模拟抽卡系统。

## 模块需求

### 1. 角色图鉴 `#characters`

- 行式列表排列，显示半身头像+名称+初始/当前修为+宗门+灵根+生命+攻击+防御+主修功法+战力
- 详情页编辑：立绘（半身头像+全身立绘）、姓名、宗门、初始修为、灵根（最多3个）
- 修为阶段系统：20个境界（练气初期~化神巅峰），每阶段独立属性（生命/攻击/防御/暴击/爆伤/五行抗性/五行增伤），默认折叠
- 被动技能（可增删）、突破I~IV阶
- 装备法宝（攻击/防具/饰品下拉选择）
- 主修功法（从功法库选择，元婴功法互斥）+ 习得神通
- 战力自动计算

### 2. 法宝图鉴 `#treasures`

- 三种类型：攻击类、防具类、饰品类
- 词条系统，品阶分级
- 行式列表+战力显示

### 3. 灵宠图鉴 `#pets`

- 行式列表：属性阶段系统+主动/被动技能+战力

### 4. 怪物图鉴 `#monsters` — 未开发

### 5. 道具图鉴 `#items`

- 行式列表：图标+名称+类型+品阶+描述
- 预置灵石（货币类），支持图片上传

### 6. 功法与神通 `#skills`

- Tab切换（功法/神通），行式列表
- 五行多选（最多3个）、等阶（人阶下/中/上）、类型（攻击/防御/增益/辅助）
- 神通额外：战技/绝技+灵力

### 7. 剧情 `#story` — 未开发

### 8. 地图 `#map`

- 从 `img/map/` 读取图片全宽展示

### 9. 伤害计算器 `#damage`

- 攻击方/防御方双栏输入+法宝选择+修为压制+五行克制+防御免伤
- 三类增伤独立乘区、技能倍率

### 10. 模拟抽卡 `#gacha-normal` / `#gacha-up` / `#gacha-skill`

- 三个卡池：寻仙唤灵（角色常驻）、仙途引渡（角色UP）、天书密阁（功法神通）
- 自定义卡池：版本号、名称、稀有度分档（红UR/金SSR/紫SR/蓝R四档可配）、奖励配置、UP配置
- 仙缘值/悟道值累进概率提升+十连保底金色+硬保底（角色100/技能60）
- 抽卡动画：单抽点击开门+十抽跑马灯+逐一开门+跳过汇总
- 灵石道具集成
